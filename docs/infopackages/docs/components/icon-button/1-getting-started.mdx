# Getting started

React component representing an custom element with editable contents.

## Quick started

### Basic Icon Button

The simplest `IconButton` component renders a icon button based on the material design 3 guidelines.

<IconButton icon={eye} /><br /><br />

```tsx
import { IconButton } from '@infotition/icon-button';

const Button: FunctionComponent = () => {
  return <IconButton icon={eye} />;
}

export default Button;
```

### Handling clicks

export const ClickDemo = () => {
  const [count, setCount] = useState(0);
  const handleClick = () => {
    setCount(count + 1);
  }
  return (
    <div style={{display: "flex", alignItems: "center"}}>
      <IconButton icon={eye} onClick={handleClick} />
      <span>Clicks: {count}</span>
    </div>
  );
}

The `IconButton` accepts an `onClick` handler which gets called after button is clicked (only if the button is not disabled).

<ClickDemo /><br />

```tsx
import { IconButton } from '@infotition/icon-button';

const Button: FunctionComponent = () => {
  const handleClick = () => {
    console.log("click");
  }

  return <IconButton icon={eye} onClick={handleClick} />;
}

export default Button;
```

### Disabled

The `IconButton` can be disabled. This disables all click actions and animations.

<IconButton icon={eye} disabled /><br /><br />

```tsx
import { IconButton } from '@infotition/icon-button';

const Button: FunctionComponent = () => {
  return <IconButton icon={eye} disabled />;
}

export default Button;
```

### Size

The `IconButton` comes in three different sizes (small, medium, large). Medium is the default size.

<IconButton icon={eye} size="large" /><br /><br />

```tsx
import { IconButton } from '@infotition/icon-button';

const Button: FunctionComponent = () => {
  return <IconButton icon={eye} size="large" />;
}

export default Button;
```

## Installation

```bash
npm install @infotition/icon-button
```

Or:

```bash
yarn add @infotition/icon-button
```

If you want the default styling to be enabled, you must import the css / scss file first.
