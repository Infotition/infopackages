# Getting started

Lightweight react button based on material design 3.

## Quick started

### Basic Button

The simplest `Button` component comes in three different variants: `contained` (default),
`outlined` and `text`. All variants can be combined with other props.

<div style={{ display: "flex", gap: "30px" }}>
  <Button variant="text" label="TEXT" />
  <Button variant="contained" label="CONTAINED" />
  <Button variant="outlined" label="OUTLINED" />
</div>
<br />

```tsx
<Button variant="text" label="TEXT" />
<Button variant="contained" label="CONTAINED" />
<Button variant="outlined" label="OUTLINED" />
```

### Form props

The standard form attributes are supported e.g. `disabled`, `type` and so on.

<div style={{ display: "flex", gap: "30px" }}>
  <Button variant="text" label="TEXT" disabled />
  <Button variant="contained" label="CONTAINED" disabled />
  <Button variant="outlined" label="OUTLINED" disabled />
</div>
<br />

```tsx
<Button variant="text" label="TEXT" disabled />
<Button variant="contained" label="CONTAINED" disabled />
<Button variant="outlined" label="OUTLINED" disabled />
```

### Contained Button

You can remove the elevation of the contained button with the `disableElevation` prop.

<div style={{ display: "flex", gap: "30px" }}>
  <Button variant="contained" label="CONTAINED" disableElevation />
</div>
<br />

```tsx
<Button variant="contained" label="CONTAINED" disableElevation />
```

### Handling Clicks

The button component accepts an `onClick` handler that is applied to the root DOM element
and fires after a user clicked the button while not `disabled`.

export const ClickDemo = () => {
  const [clicks, setClicks] = useState(0);
  const handleClick = () => {
    setClicks(clicks + 1);
  }
  return (
    <div style={{ display: "flex", gap: "20px", alignItems: "center" }}>
      <Button variant="contained" label="CONTAINED" onClick={handleClick} />
      <span>Clicks: {clicks}</span>
    </div>
  );
}

<ClickDemo /><br />

```tsx
<Button
  variant="contained"
  label="CONTAINED"
  onClick={() => alert('clicked')}
/>
```

### Icons

Buttons can contain a icon at the start or end of the button.

<div style={{ display: "flex", gap: "30px" }}>
  <Button variant="contained" label="PRIMARY" startIcon={bolt} />
  <Button variant="contained" label="SECONDARY" endIcon={eye} />
  <Button variant="contained" label="PRIMARY" startIcon={bolt} endIcon={eye}  />
</div><br />

```tsx
<Button variant="contained" label="PRIMARY" startIcon={bolt} />
<Button variant="contained" label="SECONDARY" endIcon={eye} />
<Button variant="contained" label="PRIMARY" startIcon={bolt} endIcon={eye}  />
```

### Loading state

The button can also show a loading state and disable interaction. If no `loadingPosition` is defined,
the `label` gets replaced by the spinner.

<div style={{ display: "flex", gap: "30px" }}>
  <Button variant="contained" label="PRIMARY" loading disabled />
  <Button variant="contained" label="PRIMARY" loading disabled loadingPosition="start" />
  <Button variant="contained" label="PRIMARY" loading disabled loadingPosition="end" />
</div><br />

```tsx
<Button variant="contained" label="PRIMARY" loading disabled />
<Button variant="contained" label="PRIMARY" loading disabled loadingPosition="start" />
<Button variant="contained" label="PRIMARY" loading disabled loadingPosition="end" />
```

Instead if the spinner, you can also provide a `loadingIndicator`, which replaces the label and the spinner.

<div style={{ display: "flex", gap: "30px" }}>
  <Button variant="contained" label="PRIMARY" loading disabled loadingIndicator="Loading..."/>
</div><br />

```tsx
<Button variant="contained" label="PRIMARY" loading disabled loadingIndicator="Loading..."/>
```

## Installation

```bash
npm install @infotition/react-button
```

Or:

```bash
yarn add @infotition/react-button
```

If you want the default styling to be enabled, you must import the css / scss file first.
